# CyborgFX

A modern Windows 11 replacement for the Mad Catz Cyborg amBX lighting software.

## Goal
- Windows 11 support
- Ambient screen sync
- Razer Chroma integration (if possible)
- Modern UI
- Open source

## Hardware
- Device: Mad Catz Cyborg amBX Gaming Lights
- VID: 06A3
- PID: 0DC5
